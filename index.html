MailyDaily

Your daily email assistant

Welcome to MailyDaily

MailyDaily is a cross-platform mobile app designed to help you manage your Gmail inbox with ease. Built using React Native, MailyDaily provides a daily summary of your emails and offers conversational assistance for managing your inbox efficiently.

Features

  • Daily Email Summaries: Get a concise summary of your new emails each morning.
  • Action Suggestions: Receive actionable suggestions such as replying, archiving, or scheduling tasks.
  • Voice Assistance: Interact with your inbox using voice commands and responses.
  • Privacy Focused: Your data is processed locally to ensure your privacy.

Getting Started

  1. Clone the Repository: 
    git clone https://github.com/yourusername/MailyDaily.git
  2. Navigate to the Project Directory: 
    cd MailyDaily
  3. Install Dependencies: 
    npm install
  4. Run the App: 
    npx react-native run-android or npx react-native run-ios
